# AI Coding Tool Research: Kilo Code

## (1) Core Architecture
Kilo Code is an open-source AI coding agent designed as a VS Code extension. Its architecture is built for "agentic engineering," where the AI is treated as a persistent collaborator rather than a one-off assistant.

## (2) Multi-agent Capabilities
Kilo features specialized **Agent Modes** that allow developers to switch contexts (e.g., from frontend development to DevOps) without losing the thread of the conversation. It is designed to run "around the clock," suggesting a move toward autonomous background task execution.

## (3) LLM Provider Support
Kilo Code boasts support for **500+ models**, likely through integration with a unified API provider like OpenRouter. This gives users extreme flexibility in choosing models based on performance, cost, or privacy requirements.

## (4) Key Innovative Features
- **ClawBytes**: A unique feature for creating custom, reusable AI workflows. "ClawBytes" act like macros or specialized scripts that the agent can execute for repetitive tasks.
- **Autonomous Workflow Focus**: Built to automate entire workflows rather than just generating snippets.
- **Inline Autocomplete**: Combines agentic chat capabilities with standard IDE features like intelligent code completion.

## (5) Plugin/Extension System
The primary extension mechanism is through **ClawBytes**, which allows the community and individual developers to share and reuse specialized agent behaviors and tool-use patterns.
