# Costrict

## Architecture Patterns
- **Quality Guardian Architecture:** A multi-layered system designed to enforce code quality and architectural standards.
- **Knowledge Base Centric:** Generates a project-specific knowledge base "optimized for AI reading" to ensure the AI understands the business logic and constraints.
- **Governance-First:** Built with the philosophy that AI tools must be governable and explainable to be approved in enterprise settings.
- **Private Deployment Focus:** Architected for on-premises or private VPC deployment to protect intellectual property.

## Key Features
- **Strict Quality Control:** Unlike other tools that prioritize speed, Costrict focuses on "strict" adherence to standards and quality metrics.
- **Architecture Guardian:** A specialized module that enforces clean architecture principles and module decoupling.
- **ADR Enforcement:** Directly supports and enforces Architecture Design Requirements (ADRs) throughout the codebase.
- **Unified Assistant & Reviewer:** Combines code generation with an aggressive, standard-driven review process.

## How They Handle Agents
- **Skill-Based Agents:** Uses a "skills" framework where agents are equipped with specific architectural and quality-enforcement capabilities.
- **Compliance Agents:** Specialized agents that check code against enterprise security and compliance policies in real-time.
- **Context-Aware Orchestration:** Agents use the "AI-optimized" knowledge base to maintain high precision in their suggestions.

## LLM Support
- **Model-Agnostic Core:** Can work with various high-reasoning models (Claude, GPT, Qwen) but is optimized for "strict" instruction following.
- **Focus on Local Models:** Often deployed with local LLMs to satisfy enterprise security requirements.

## Unique Innovations & Differentiators
- **"Strictness" as a Feature:** While most AI tools are "helpful," Costrict is designed to be a "gatekeeper" for quality, rejecting code that doesn't meet standards.
- **AI-Readable Documentation:** Automatically generates and maintains documentation specifically designed to help other AI agents understand the codebase.
- **Enterprise Governance:** Provides a management dashboard for leads to monitor and control how AI is being used and what standards it is enforcing.
