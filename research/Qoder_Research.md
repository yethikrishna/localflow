# AI Coding Tool Research: Qoder

## (1) Core Architecture
Qoder is an agentic AI coding platform that provides both an **AI-Native IDE** and a **CLI tool**. It is designed to be a comprehensive "Agentic IDE" where the agent is a first-class citizen in the development environment, similar to Cursor or Antigravity.

## (2) Multi-agent Capabilities
The platform is built around "Agentic Workflows." It focuses on **Conversation-based Programming**, where the developer collaborates with the agent to build entire software systems. It handles multi-file changes and project-wide context better than standard chat-based extensions.

## (3) LLM Provider Support
Qoder typically offers a unified experience, likely leveraging top-tier models like GPT-4o or Claude 3.5 Sonnet. It is focused on providing a "real software development" experience through highly capable models.

## (4) Key Innovative Features
- **Unified IDE/CLI**: Seamless transition between a visual IDE experience and a powerful terminal-based agent.
- **End-to-End Development Focus**: Positions itself not just as a code completer, but as a platform for building production-ready software.
- **Agentic IDE Experience**: Deeply integrated UI elements that allow the agent to present plans and get feedback visually.

## (5) Plugin/Extension System
Qoder mentions a system for extending its agentic capabilities with custom tools, though specific details on a public protocol like MCP are less prominent than in Cline or RooCode.
