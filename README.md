## 🌌 The Narrative
In an era where AI coding is dominated by cloud-locked IDE extensions and "God-model" silos, **LocalFlow** returns sovereignty to the developer. 

LocalFlow is not a plugin. It is a **parallel agentic development environment** built from the ground up for speed, privacy, and massive orchestration. It allows you to plan and execute complex engineering missions with an army of specialized agents running simultaneously across local and remote compute.

Powered by the **NEXUS Engine**, LocalFlow bypasses the decade-old bottlenecks of the DOM and CSS, rendering high-density agentic data at 60fps directly to a 2D Canvas.

---

## ⚡ Core Pillars

### 1. True Parallelism (NexusMux)
Most AI tools are single-threaded conversations. LocalFlow is a multiplexer. Spawn N specialized agents (Auditor, Implementer, Researcher) in isolated Git worktrees. Conflict detection and merge simulation are built into the core.

### 2. Zero-CSS Architecture (NexusCanvas)
Built on `@chenglou/pretext` and Yoga Layout (WASM), our rendering engine is **300-600x faster** than standard DOM measurement. No reflows. No layout shifts. Just pure performance for the high-octane data streams of the agentic era.

### 3. Provider Sovereignty (NexusLLM)
100% provider-agnostic. Whether it's **Ollama** running locally on your 4090, **OpenRouter** aggregating the world's best models, or direct-to-metal **Claude/GPT-4** access—you own the inference.

### 4. The Model Context Bridge (NexusTools)
A unified MCP (Model Context Protocol) bridge that connects your agents to everything: Browser Use, STORM, AppFlowy, NocoDB, and your local filesystem.

---

## 🛠️ Technical Stack

- **Orchestration**: TypeScript Monorepo (Turbo/pnpm)
- **Rendering**: Pretext + Yoga (Flexbox) + Canvas 2D
- **Discovery**: Docker-socket-based Auto-Discovery for local tools
- **Memory**: 3-Tier Persistent Memory (Episodic, Semantic, Procedural)
- **Deployment**: Production-ready Docker stack with GPU support

## 🚀 Getting Started

LocalFlow is currently in its **Month 2: Build Complete** state. You can launch the full environment with a single command.

### Prerequisites
- Docker & Docker Compose
- Node.js & pnpm
- (Optional) Ollama for local inference

### Installation
```bash
git clone https://github.com/yethikrishna/localflow.git
cd localflow
bash scripts/setup.sh
```
